Schematic

This is the schematic for the A2 assignment. I picked the value 220 for the resistor because I used the equation 5V - 1.8V = (20*10^-3)R. 5V represents the initial voltage and 1.8 represents the voltage drop. (20*20^-3) represents the 20mA desired current. I also used this equation for the button's resistor and got the closest resistor value to 220 ohms.

Arduino Board Photo

This is Arduino Board setup I used to get my LEDs to fade. I purposely set separate input wires for each LED while having a 5V power source connected to the beginning of the circuit. This allowed me to be able to code each LED separately.
