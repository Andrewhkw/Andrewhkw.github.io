Schematic

This is the schematic for the A3 assignment. I picked the value 220 for the resistor because I used the equation 5V - 1.8V = (20*10^-3)R. 5V represents the initial voltage and 1.8 represents the voltage drop. (20*20^-3) represents the 20mA desired current. I also used this equation for the button's resistor and got the closest resistor value to 220 ohms. I chose the 10k resistor because resistance is variable and either 5k or 10k resistors could be used.

Arduino Board Photo

This is Arduino Board setup I used to get my LEDs to fade with a photo resistor. I purposely set separate input wires for each LED while having a 5V power source connected with an A5 pin controlling current. This also allowed me to be able to code each LED separately.
